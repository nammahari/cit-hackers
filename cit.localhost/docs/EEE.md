# Electrical and Electronics Engineering (EEE) Curriculum

## Semester 1
* Heritages of Tamil
* Mathematics-I (Calculus and Linear Algebra)
* Communicative English - I
* Engineering Physics
* Electric Circuit Analysis
* Sensors and Instrumentation Systems
* Problem Solving using C Programming
* Employability Enhancement Skills - I

## Semester 2
* Mathematics-II (Probability and Statistics)
* Tamils and Technology
* Electro Magnetic Field Theory
* Communicative English - II
* Electronic Devices and Circuits
* Digital Logic Circuits
* Data Structures using C Programming
* Employability Enhancement Skills - II

* Signals and Systems
* DC Machines and Transformers
* Analog Integrated Circuits
* Microprocessor and Microcontroller
* Object Oriented Programming in JAVA
* Employability Enhancement Skills - III
* Core Course Project - I

## Semester 4

* Transmission and Distribution
* Advanced Database Management System
* Synchronous and Induction Machines
* Control Systems Engineering
* Power Electronics
* Employability Enhancement Skills - IV

## Semester 5
