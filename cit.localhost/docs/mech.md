# Third Semester Subjects

---

## 1. Data Structures

### Overview
Data Structures focuses on organizing and storing data efficiently so that programs run faster and use less memory.

### Topics Covered
- Arrays and Strings
- Linked Lists
- Stacks and Queues
- Trees and Graphs
- Searching and Sorting

### Importance
This subject builds strong problem-solving skills and is essential for coding interviews and advanced programming.

### Learning Outcomes
- Understand common data structures
- Apply the right structure to a problem
- Write efficient programs

---

## 2. Object Oriented Programming (OOPS)

### Overview
OOPS teaches how to design programs using objects and classes, making code reusable and easy to maintain.

### Topics Covered
- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction

### Importance
OOPS is widely used in real-world software development and is the foundation of modern programming languages.

### Learning Outcomes
- Design programs using OOP concepts
- Build modular and reusable code
- Understand real-world software design

---

## 3. Database Management Systems (DBMS)

### Overview
DBMS explains how data is stored, managed, and retrieved efficiently using databases.

### Topics Covered
- Database concepts
- SQL queries
- Normalization
- Transactions
- Indexing

### Importance
Almost every application uses databases, making this subject critical for backend development.

### Learning Outcomes
- Design simple databases
- Write SQL queries
- Manage and secure data

---

## 4. Computer Organization and Architecture

### Overview
This subject explains how a computer system works internally, from hardware components to instruction execution.

### Topics Covered
- CPU architecture
- Memory organization
- Input and Output systems
- Instruction cycles
- Basic pipeline concepts

### Importance
Understanding hardware improves software performance and system-level programming.

### Learning Outcomes
- Understand how computers process instructions
- Identify hardware components and their roles
- Analyze system performance at a basic level

---

## 5. Discrete Mathematics

### Overview
Discrete Mathematics provides the mathematical foundation required for computer science problem-solving.

### Topics Covered
- Logic and Proofs
- Sets and Relations
- Graph Theory
- Combinatorics
- Boolean Algebra

### Importance
This subject supports algorithms, data structures, and computer networks.

### Learning Outcomes
- Apply mathematical logic to computing problems
- Analyze graphs and relations
- Improve analytical thinking

---

## 6. Software Engineering

### Overview
Software Engineering teaches how to plan, design, develop, and maintain software systems systematically.

### Topics Covered
- Software Development Life Cycle (SDLC)
- Requirement Analysis
- Software Design
- Testing methods
- Maintenance

### Importance
This subject helps in building reliable, scalable, and maintainable software systems.

### Learning Outcomes
- Follow structured software development processes
- Understand teamwork and documentation
- Deliver quality software solutions
