    Computer Science and Engineering (CSE)

    Semester-wise Subjects (R24 Regulation)

    Semester I

    • Induction Programme
    • Mathematics – I (Calculus and Linear Algebra)
    • Heritage of Tamils
    • Communicative English – I
    • Professional Ethics and Universal Human Values
    • Engineering Physics
    • Digital System Design
    • C++ Programming
    • Employability Enhancement Course – I

    Semester II

    • Mathematics – II (Probability and Statistics)
    • Tamils and Technology
    • Communicative English – II
    • Web Development Essentials
    • Data Structures using C++
    • Database Management Systems
    • Introduction to Java Programming
    • Employability Enhancement Course – II
    • NCC Credit Course – Level 1 (Optional)

    Semester III

    • Discrete Mathematics
    • Operating Systems
    • Design and Analysis of Algorithms
    • Web Development Frameworks and Practices
    • Artificial Intelligence
    • Python Programming
    • Core Course Project – I
    • Employability Enhancement Course – III
    • Mandatory Course – I

    Semester IV

    • Theory of Computation
    • Computer Architecture
    • Machine Learning
    • Introduction to Cyber Security
    • Software Engineering
    • Computer Networks
    • Core Course Project – II
    • Employability Enhancement Course – IV
    • Mandatory Course – II
    • NCC Credit Course – Level 2 (Optional)

    Semester V

    • Compiler Design
    • Embedded Programming
    • Data Analytics and Visualization
    • Professional Elective – I
    • Professional Elective – II
    • Professional Elective – III
    • Core Course Project – III
    • Employability Enhancement Course – V

    Semester VI

    • Managerial Economics and Financial Accounting
    • Distributed Computing
    • Software Testing
    • Professional Elective – IV
    • Professional Elective – V
    • Open Elective – I
    • Core Course Project – IV

    Semester VII

    • Professional Elective – VI
    • Open Elective – II
    • Project Phase – I
    • Internship

    Semester VIII

    • Project Phase – II

    Notes

    • Professional Electives (PEC) and Open Electives (OE) are chosen based on specialization.
    • NCC courses are optional and not counted towards CGPA.
    • Curriculum follows Regulation 2024 (R24).
